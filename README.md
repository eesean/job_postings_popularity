# Predicting Job Postings Popularity

Designed a multi-agent system to predict level of engagement of a LinkedIn job posting leveraging PySpark for data processing, comprising of the following agents:

1. Adaptive Threshold Agent
- Dynamically determines thresholds for engagement categorisation

2. Exploratory Data Analysis Agent
- Perform exploratory data analysis to understand distributions, missing values, and feature relationships

3. Feature Engineering Agent
- Clean and transform raw data into model-ready features

4. Machine Learning Agent
- Prepares model pipelines and integrates preprocessing with modelling

5. Evaluation Agent
- Train models and evaluates model outputs and feature importance, thereafter generating a summarised report
